
Problem Statement:

Create the application which contains two services named as Number and String. Number service contains one method named as ChkPrime() which accepts number and check wheather that number is prime or not. String service contains one method named as CountCapital() which counts the number of capital characters and return its count. We have to create two child component named as Child1 & Child2 under app component Chil1 uses Number service & Child2 uses String service using Dependancy Injection. Call both the methods from the respective components by passing some hardcoded values and display the result.
